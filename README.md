# Block Jam Master 🚌🧩

A high-energy, strategic block-sorting puzzle game built with React, Vite, and Tailwind CSS.

## 🎮 How to Play
- **Sort the Blocks**: Tap on free blocks to move them to the dock.
- **Fill the Bus**: Match 3 blocks of the same color to load them onto the waiting bus.
- **Clear the Board**: Fill all buses before the timer runs out to win the level!
- **Watch Out**: Don't let your dock fill up (7 slots), or it's game over!

## ✨ Features
- **Hard Mode Mechanics**:
  - ❄️ **Frozen Blocks**: Tap once to unfreeze before moving.
  - ❓ **Mystery Blocks**: Colors are hidden until they reach the top.
- **Combo System**: Match blocks in quick succession for score multipliers and bonus coins.
- **Daily Rewards**: Log in daily to claim 100 bonus coins.
- **In-Game Shop**: Buy time extensions or clear blocks from your dock.
- **Revive System**: Use ads or coins to get a second chance when you fail.
- **Settings**: Toggle sound and haptic feedback.

## 🚀 Tech Stack
- **Frontend**: React 18, TypeScript, Vite
- **Animations**: Motion (Framer Motion)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React

## 🛠️ Installation & Development

1. **Clone the repository**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/block-jam-master.git
   cd block-jam-master
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
